# Twitter 2025 클론코딩
- 트위터 일부 기능을 구현
  - 홈 화면에서 트윗 읽기/작성/수정
  - 프로필 화면에서 본인 트윗 확인 및 닉네임 수정
  - 로그인 화면에서 이메일 또는 깃헙 로그인
  - 로그아웃 기능
- 사용 기술 : React, TypeScript, Vite, Styled Components, Firebase
- 접속 링크 : https://twitter-2025.web.app/
